======================================
###         COMANDOS GIT           ###
======================================


# 1. Começar um projeto com git

```sh
    git init

```

# 2. Adicionando arquivos

```sh
    git add .
```

# 3. Permitindo alterações 

```sh
    git commit -m "feat: Primeira aula de git"
```

# 4. Criar um repositório nbo Github

```sh
    entre no github e crie um repositório
```

# 5. Transformando a branch master para a main

```sh
    git branch -M main
```

# 6. Checar a branch atual de trabalho

```sh
    git branch
```

# 7. Atualizando a main

```sh
    git push origin main
```
